# AxiomGraphics_Engine
AxiomGraphics Engine-это одновременно графический фреймворк и движок. Он создан для изучения основ 2D и 3D графики. Также он может использоваться в реальных проектах. Распространяется по лицензии LGPL-3.0. Проект находится в активной разработке и открыт для участия. Пока поддерживается только OpenGL.В будущем планируется поддержка других API
