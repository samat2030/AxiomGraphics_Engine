## Описание  
AxiomGraphics Engine — это одновременно графический фреймворк и движок.  
Он создан для изучения основ 2D и 3D графики. Также он может использоваться в реальных проектах.  
Распространяется по лицензии LGPL-3.0.  

Проект находится в активной разработке и открыт для участия.  

Пока поддерживается только OpenGL. В будущем планируется поддержка:  
* OpenAL  
* Vulkan   
* DirectX11  
* и другие API  
  
## Структура движка:  

```
AxiomGraphics_Engine:
  core:
    libs:
      glfw-3.4/
      glad/
      stb:
        stb_image.h
    platform:
      opengl.h 
    core.h
    state.h
  Engine.h
app.cpp
```

> Все API хранятся в директории `platform`, и каждый из них реализует первую абстракцию.  
  
> `state.h` хранит `struct` и объекты, которые должны быть доступны во всём проекте.  
  
> `core.h` собирает все файлы и создаёт вторую абстракцию.  
  
> `Engine.h` предоставляет пользователю множество инструментов и инструкций.  
> Его можно использовать новичкам, которые не хотят разбираться с низкоуровневой частью.  
> Также он содержит `встроенные утилиты`, например `настройку UV`.  
  
> Те, кто хочет работать на более низком уровне, могут через `Engine.h` получить доступ к `core.h`.  
  
> `app.cpp` — это файл пользователя.  
  
> Директория `libs` хранит `сторонние библиотеки`, например `GLFW`.
